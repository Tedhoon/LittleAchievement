## ✔ Commit Message Format 

- [add] 추가한 내용
- [mdf] 수정한 내용
- [del] 지운 내용
- [merge] 병합

```bash
ex) main app 생성 시 
$ git commit -m "[add] main app 생성"
```

## ✔ 협업 가이드라인 

`[개발 단계]`

1. 각자의 branch에서 작업
2. 각자의 branch에서 develop branch에 Pull Request
3. 최소 1명 검열 후 develop branch에서 모든 개발 코드 merge 

`[배포 단계]`

4. develop branch에서 master에 MERGE
