# 규약

# Commit Message Format
- [add] 추가한 내용
- [mdf] 수정한 내용
- [del] 지운 내용
- [merge] 병합

> 예시 : [add] main app 생성

# branch
- 루트 및 배포 : master
- 개발 : develop
- 개인 : 각자 닉네임 or 이름

# 협업 순서

[개발 단계]
1. 각자의 branch에서 작업
2. 각자의 branch에서 develop에 Pull Request
3. develop branch에서 최소 1명 검열 후 merge 

[배포 단계]

4. develop branch에서 master에 Pull Request
5. master branch에서 merge
